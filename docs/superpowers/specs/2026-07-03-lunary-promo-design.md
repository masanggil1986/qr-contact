# Lunary 프로모 카드 — 설계 문서

작성일: 2026-07-03

## 목적

QR 생성기 페이지 하단에 제작사(Lunary) 개발 의뢰 광고를 붙인다. 골자: 대한민국
개발팀(북한 아님 — 유머러스한 윙크 톤), SvelteKit 주력 + Flutter/bun, 영어 소통
가능, 개발자 직접 고용보다 저렴, 유지보수는 의뢰사 내부 직원이 이어받도록 깔끔한
코드로 전달. 링크: https://www.lunary.ai.kr

## 핵심 결정

- **톤**: 유머러스한 윙크 — "(네, 북한 아닙니다 🙂)"
- **배치**: 폼/QR 그리드 아래, 푸터 위. 가로 전체 폭 프로모 카드 1장.
- **형태**: 단일 정적 컴포넌트 `src/lib/components/LunaryPromo.svelte`
  (로직 없음, 기존 컴포넌트 구조와 일관).

## 문구 (확정)

- 배지(모노, 기존 `vCard 3.0` 배지 스타일): `MADE BY LUNARY`
- 제목: **웹·앱 개발이 필요하신가요?**
- 본문:
  - "이 사이트는 대한민국 개발팀 **Lunary**가 만들었습니다. (네, 북한 아닙니다 🙂)"
  - "이 명함 QR 생성기는 가벼운 데모일 뿐이에요. 실제로는 훨씬 복잡하고 어려운 웹
    서비스와 앱을 개발합니다 — SvelteKit 주력, Flutter 앱, bun 기반이고 영어로도
    편하게 소통할 수 있어요."
  - "개발자를 직접 고용하는 것보다 저렴하게 개발하고, 유지보수는 내부 직원분들이
    바로 이어받을 수 있도록 깔끔한 코드로 전달합니다."
- 기술 칩 4개(모노, 슬레이트 톤): `SvelteKit` `Flutter` `Bun` `English OK`
- CTA(블루 솔리드 버튼): "프로젝트 문의하기 →" → https://www.lunary.ai.kr
  새 탭(`target="_blank"`, `rel="noopener noreferrer"`)

## 디자인

기존 시스템 그대로: `rounded-2xl border border-slate-200 bg-white`, 블루 단일
액센트, 그리드와 같은 컨테이너 폭. 데스크톱은 텍스트 좌측/CTA 우측, 모바일은
세로 스택. 그라데이션·보라색·그림자 금지(기존 규칙).

## 테스트

정적 마크업(로직 없음) → 단위 테스트 없음. 검증: svelte-autofixer 클린,
`bun run check` 0 에러, 기존 e2e 회귀 통과, 브라우저 시각 확인.
